# 💰 Expense Tracker

A Progressive Web App (PWA) for tracking personal expenses. Works completely offline and can be installed on any device.

## 🚀 Live App

**[Open Expense Tracker](https://hxnism.github.io/expense-tracker-app/)**

## ✨ Features

- 💰 **Track Expenses** - Add, edit, and delete expenses
- 📊 **Categories** - Organize by Food, Transport, Shopping, Bills, Entertainment, Health, and more
- 📈 **Statistics** - View spending by category with visual charts
- 📅 **Time Filters** - Filter by today, this week, or this month
- 💾 **Offline Storage** - All data stored locally on your device
- 📱 **Install as App** - Works like a native app on Android/iOS
- 🔒 **Privacy First** - No servers, no tracking, all data stays on your device

## 📱 Installation

### On Android (Chrome)
1. Open the app URL in Chrome
2. Tap the menu (⋮) → "Install app" or "Add to Home Screen"
3. Confirm installation
4. App icon appears on your home screen!

### On iOS (Safari)
1. Open the app URL in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
4. Confirm

### On Desktop (Chrome/Edge)
1. Open the app URL
2. Click the install icon (⊕) in the address bar
3. Click "Install"

## 🎯 How to Use

1. **Add Expense** - Tap the + button
2. **Select Category** - Choose from 7 categories
3. **Enter Details** - Amount, description, and date
4. **Save** - Your expense is saved locally
5. **View Stats** - Check spending breakdown by category
6. **Export Data** - Download your expenses as JSON

## 💾 Data Storage

All expenses are stored in your browser's localStorage:
- ✅ Works completely offline
- ✅ No internet required
- ✅ Data never leaves your device
- ✅ Export anytime as backup

## 🛠️ Technical Stack

- Pure HTML5, CSS3, JavaScript
- Progressive Web App (PWA)
- Service Worker for offline support
- LocalStorage for data persistence
- Chart.js for visualizations

## 📂 Files

- `index.html` / `expense_tracker.html` - Main app
- `manifest.json` - PWA configuration
- `sw.js` - Service worker for offline support

## 🔄 Updates

The app automatically updates when you visit the URL. To force an update:
- Pull down to refresh (mobile)
- Hard reload (Ctrl+Shift+R on desktop)
